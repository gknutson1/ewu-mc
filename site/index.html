This modpack is formatted as a CurseForge pack. This guide shows how to use Prism Launcher to install and manage the modpack, but any Minecraft launcher that supports CurseForge modpacks, such as the CurseForge launcher or MultiMC, should work. You can download the latest version of the pack from the CurseForge page or GitHub repo.

This guide assumes you have zero experience with modded Minecraft and aims to be as detailed as possible. If you have some experience with modded Minecraft, you likely only neeed to skim this guide or not need to use it at all.

System Requirements

To join the Server, you must have a computer capable of running Minecraft Java Edition 1.12.2. In general, you will need a computer with a multicore x86-64 processor, running either Windows, Linux, or MacOS, although other OSes may work. It is reccommended to give the modpack at least 4 GB of RAM, so ideally you should have at least 8 GB of RAM. However, the modpack can run on computers with as little as 6 GB of RAM. This tutorial is primarily for Windows users, so users of other OSes may need to modify these instructions as they see fit.

Installing Java

This modpack requires Java 8 to run. If you do not have Java 8 installed, you can download the most recent Java 8 update from java.com/download.

Installing Prism Launcher

  • Go to prismlauncher.org/download and download the version for your operating system.
  • Run the installer and begin following the instructions until you encounter a page asking you to set a java version.
  • Select the first line with a Version number of 1.8.0_XXX (where XXX is any number).
  • Set the Maximum Memory Allocation box to the appropriate ammount based on the ammount of RAM your computer has:
    • You can see how much RAM you have by opening Settings and selecting System > About. Move to the Device Specifications section and look for the Installed RAM line - You should see an output similar to 8.00 GB (7.59 usable). IGNORE the usable part - use the first number only.
    • Multiply that first number by 1024 to get the number of megabytes of RAM your computer has.
    • Use the below table to determine the approprate value to put in the Maximum Memory Allocation box:
      • Installed RAM Maximum Memory Allocation
      Less than 6144 MiB NOT SUPPORTED
      6144 to 10240 MiB ( Installed RAM / 2 ) + 4096 MiB
      More than 10240 MiB 8192 MiB
  • Complete any additional installation steps.
  • Once the installation has been completed, you will find yourself on the main PrismLauncher window.
  • Select the Accounts drop down in the top right section of the screen, and select the Manage Accounts... button
  • A new window will appear. Select the Add Microsoft button in the top right section of the window, and complete any prompts given to you by the program.
  • Close the accounts window using the button in the lower right corner of the window.

Installing the Modpack

  • So to curseforge.com/minecraft/modpacks/ewu-mc and select the Install button.
  • If a prompt appears asking for permission to open the link with Prism Launcher, allow it.
  • A window should appear in PrismLauncher, allowing you to name the instance. Select the OK button in the bottom right.
  • Prism Launcher will now install all mods The modpack should now be installed.
