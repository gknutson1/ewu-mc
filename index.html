If you wish to contribute, you can find the source code at the 'site' branch at github.com/gknutson1/ewu-mc. You can also find the source code for the modpack there if you want to roll your own version of the pack or contribute to improving the pack.

The server has a Discord! The discord is where we announce updates, coordinate events, and talk with other players. Join at discord.gg/TE2bvDM5. If you have any problems or questions about the site, the modpack or the server, contact me on Discord by pinging me (@Gunnar Knutson) in the #server-help channel on the Discord.

This modpack is formatted as a CurseForge pack. This guide shows how to use MultiMC to install and manage the modpack, but any Minecraft launcher that supports CurseForge modpacks should work. You can download the latest version of the pack from the GitHub repo.

This guide assumes you have zero experience with modded Minecraft and tries to be as detailed as possible. If you have some experience with modded Minecraft or have used MultiMC before, you may only neeed to skim this guide or not need to use it at all.

System Requirements

To join the Server, you must have a computer capable of running Minecraft Java Edition 1.12.2. In general, you will need a computer with an x86-64 processor, running either Windows, Linux, or MacOS, although other OSes may work. It is reccommended to give the modpack at least 4 GB of RAM, so ideally you should have 8 GB of RAM. However, the modpack can run on computers with as little as 4 GB of RAM. This tutorial is primarily for Windows users, so users of other OSes may need to modify these instructions as they see fit.

Installing Java

The server requires Java 8, preferrably the most recent version (8u311 at the time of writing). To install Java, go to java.com/en/download and select "Agree and Start Free Download". Once the download has completed, Run the installer and follow all of it's prompts.

Installing MultiMC

Go to multimc.org and scroll down until you find the "downloads" section. Select the button marked "Download (32/64-bit)" under the Windows logo. This will download a file labled "mmc-stable-win32.zip". Extract the files in this folder to wherever you wish to install minecraft to. Then start MultiMC by opening the folder you just extracted, and starting the program "MultiMC.exe".

When you first open MultiMC, you may get a warning from Windows SmartScreen saying that the program is not signed. Simply click "more info" and then "Run Anyway". MultiMC will then ask you to select your language. Do so. Next, you need to select your Java version and memory allocation settings. For java, select a version of java that has an architecture of 64 and a version number of 1.8.0_xxx, where the x's are any number. For memory allocation, insert the ammount of ram you wish to assign to the modpack in the "Maximum memory allocation" field, in MB. See the table below for the reccommendations, based on how much RAM is in your computer. You can see how much RAM you have in your computer by going to settings -> system -> about and looking for the 'Installed RAM' field.

System RAM    Memory Allocation
   More than 8 GB    6 GB (6144 MB)
   6 to 8 GB    4 GB (4096 MB)
   4 to 5 GB    3 GB (3072 MB)
   Less than 4 GB    Not reccommended

Once you have set the memory allocation, select "Next". MultiMC will ask you if you wish to Enable Analytics. You can chose to enable or disable Analytics, and then click "Finish".

Signing In

Now that you are in MultiMC, select the button marked "Settings" at the top of the MultiMC UI, and then select "Accounts" in the window that pops up.

 Mojang Account

If you have a Mojang account (you bought Minecraft through mojang.com and have NOT yet migrated your account), select "Add Mojang", input your username and password, and hit OK.

 Microsoft Account

If you have a Microsoft account (you bought Minecraft through Minecraft.net, or migrated your Mojang account), Select "Add Microsoft". A window will appear with a URL and a 8 digit code. Open the URL and input the 8 digit code into the website you just opened. In the website, click next, and then follow the prompts to sign in to your Microsoft account. A message will appear in the site asking you to grant MultiMC access to your Microsoft account. Select yes to finish signing in.

Installing the Modpack

Go to github.com/gknutson1/ewu-mc/releases/latest and download the file labled ewu_mc_client_X.X.X.X_RX.zip, where X is some number. Go back to MultiMC and select "Add Instance" in the upper left corner of the window. In the window that pops up, select "import from zip", and then "browse". Slelect the zip file you just downloaded, and then select "ok". MultiMC will take a minute to unpack the folder, and you will then see the modpack in the main multiMC screen. Now you can simply hit the "launch" button on the right side of the screen to start playing!

Updating the Modpack

Delete the current install of the modpack by right-clicking on it and selecting "delete", then "yes". Then proceed to Installing the Modpack.
